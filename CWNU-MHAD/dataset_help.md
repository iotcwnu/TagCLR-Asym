CWNU-MHAD is a dataset constructed for multimodal Human Activity Recognition (HAR) tasks. By simultaneously capturing RFID signals and RGB video data, this dataset records changes in wireless signals and visual motion information during various human activities. It can be used for research in RFID monomodal recognition, visual monomodal recognition, and RFID-visual multimodal fusion recognition.

Data collection was conducted in a laboratory environment. The RFID acquisition system consists of an Impinj Speedway R420 reader, a Laird S9028PCR antenna, and 16 E41C Impinj passive RFID tags. These 16 RFID tags were deployed on key body parts of the volunteers—such as the wrists, elbows, knees, and ankles—to simulate the nodes of a human RFID skeleton and capture signal changes during activity. The reader operates in the 920 MHz–925 MHz frequency band; the experiment used the 925 MHz channel with a transmission power set to 32.5 dBm. Concurrently, the system used a webcam to synchronously capture RGB video at a resolution of 640×480 and a frame rate of 30 FPS.

During data acquisition, the RFID antenna transmits radio frequency signals and receives reflected response signals from the body tags; the reader records these response signals as RFID tag response data. Each tag response record includes the EPC, timestamp, DF, RSSI, phase, and an activity ID indicating the current activity. A total of 10 volunteers were recruited to participate in the data collection for this dataset. Each volunteer performed 21 categories of human activities, with each activity repeated approximately 40 times. A complete activity corresponds to one activity sample; therefore, the dataset contains a total of approximately 10 × 21 × 40 = 8,400 multimodal activity samples. Activity categories include standing, squatting, bending over, walking, marching, sitting, and lying down. It also includes several sets of similar movements, such as “walking with wide arm swings” and “walking with small arm swings,” “striding without arm swings” and “taking small steps without arm swings,” and “marching in place with arm swings” and “marching in place without arm swings,” which can be used to test a model’s ability to distinguish between fine-grained, similar activities.

CWNU-MHAD stores data using activity samples as the basic unit. Each sample consists of an .mp4 video file and a .txt RFID sensing data file synchronized with the video. The .mp4 file records the complete video of the volunteer performing the activity, while the .txt file records the response data from all RFID tags during that activity. The data is organized by volunteer ID, activity category ID, and number of acquisitions.

**==Important Note:==** This dataset contains recognizable facial and body appearance information of volunteers. Before downloading or using this dataset, please be sure to read and comply with the Privacy and Ethics Statement. It is strictly prohibited to use the data for facial recognition, identity identification, privacy inference, commercial surveillance, or any other purpose that may infringe upon the privacy of volunteers.
